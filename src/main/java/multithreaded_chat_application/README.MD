### Task 3: Multithreaded Chat Application
- **Description:** A real-time chat system allowing multiple clients to connect to a single server.
- **Key Concepts:** Java Sockets (`ServerSocket`, `Socket`), Multithreading (`Thread` class), and Synchronization.
- **How to run:** 1. Compile both files: `javac *.java`
    2. Run the server first: `java ChatServer`
    3. Open multiple terminals and run: `java ChatClient`
